# 2026 LoL 멸망전 with Gen.G — 사용자 매뉴얼

상단 메뉴에서 4개 화면을 이동할 수 있습니다: **티어 데이터베이스 / 팀 구성하기 / 구성된 팀 / 대결 기록**

---

## 1. 티어 데이터베이스

참가 스트리머 목록과 등급을 모아보는 화면입니다.

- **DB 등록하기**: 우측 상단 버튼으로 스트리머를 새로 등록합니다. 스트리머명, SOOP 계정 ID, 롤 게임 아이디(아이디#태그), 라인을 입력하고 "티어 확인" 버튼으로 현재 솔로랭크 티어를 조회해볼 수 있습니다.
- **보기 방식 전환**: "티어별 / 라인별" 토글로 등급 순서 또는 라인(탑/정글/미드/원딜/서포터) 순서로 목록을 다시 정렬해서 볼 수 있습니다.
- **닉네임 검색**: 검색창에 닉네임 일부만 입력해도 포함되는 스트리머만 걸러서 보여줍니다.
- **바로가기**: 등급/라인 버튼을 누르면 해당 그룹으로 화면이 스크롤됩니다.

## 2. 팀 구성하기

5개 라인(탑/정글/미드/원딜/서포터)에 스트리머를 배치해 팀을 만드는 화면입니다.
- **팀장 등록**: 본인 스트리머명을 입력하고 확인을 누르면 팀장으로 등록되고, 해당 라인에 자동 배치됩니다.
- **스트리머 풀 필터링**: 왼쪽 목록에서 닉네임 검색, 라인, 티어, 점수 구간으로 원하는 스트리머를 좁혀서 찾을 수 있습니다.
- **드래그 앤 드롭**: 왼쪽 풀에서 스트리머 카드를 끌어와 원하는 라인 자리에 놓으면 라인업에 배치됩니다. 라인이 서로 다르면 배치 전 한 번 더 확인합니다.
- **합산 점수 확인**: 배치한 스트리머들의 점수 합계와 한도(182점)를 실시간으로 보여줍니다.
  - **팀 조합 어드밴티지**: 정글·서포터 또는 원딜·서포터가 둘 다 일정 등급(S+ 이상 또는 A- 이상) 조합이면 점수에서 자동으로 할인(-5점 또는 -3점, 가장 큰 것 하나만)이 적용되어 표시됩니다.
- **추천 스트리머**: 남은 라인이 있을 때 "추천 스트리머" 버튼을 누르면 남은 점수에 맞는 후보(또는 조합)를 추천받아 바로 선택할 수 있습니다.
- **팀 빌드**: 팀명을 입력하고 확인하면 완성된 팀 구성을 저장할 수 있습니다.
- **이미지로 캡처**: 📷 버튼으로 완성된 라인업을 이미지 파일로 저장할 수 있습니다.

## 3. 구성된 팀

지금까지 저장된 팀들의 목록을 확인하는 화면입니다.

- 각 팀 카드에서 팀명, 팀장, 승/패 전적, 승률, 라인업을 확인할 수 있습니다.

## 4. 대결 기록

팀 간 대결 결과와 각 팀의 라인업을 함께 보여주는 화면입니다.

- 팀 필터 버튼으로 특정 팀이 치른 대결만 골라볼 수 있습니다.
- 각 대결 카드에서 두 팀의 스코어(예: 3:0)와 양 팀 라인업을 함께 확인할 수 있습니다.
